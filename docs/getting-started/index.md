# Getting Started

This section is the fastest path to installing `affine_mpc` and running a first example.

## First Steps

1. Choose an interface:
    - [Python](python.md) for experimentation, analysis, plotting, notebooks, and teaching workflows
    - [C++](cpp.md) for native integration, tighter runtime control, and lower-overhead deployment
1. Follow that page to install the library and run the first example.
1. Confirm that the example runs successfully and writes its expected logs.

## After Your First Run

- Read [Concepts](../concepts/index.md) to understand the supported problem class and mathematical structure.
- Read [Usage](../usage.md) to learn the normal API workflow.
- Read [Examples](../examples.md) for more detail on the mass-spring-damper walkthrough.
- Read [Logging](../logging.md) to inspect saved outputs and prediction logs.
